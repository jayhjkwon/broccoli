## Prerequisites
- Node.js v8.0+

## Quick Start
```sh
npm install
npm run dev
```

## Run test
```sh
npm test
```

## Run the application in production mode
```sh
npm start
```

- React
- Redux
- React-Router
- Styled Components
- Jest
- Enzyme
